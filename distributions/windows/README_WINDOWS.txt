Book Triage - Windows Installation and Usage Guide
==================================================

QUICK START:
1. Double-click 'install.bat' to install dependencies
2. Double-click 'book_triage.bat' or 'start.bat' to run
3. Open http://localhost:8000 in your browser

REQUIREMENTS:
- Windows 10/11
- Python 3.8 or higher
- Internet connection for AI features

TROUBLESHOOTING:
- If you see "Python is not installed", download Python from python.org
- If the database appears empty, the app will use sample data or create a new database
- Make sure to allow the application through Windows Firewall if prompted

FILES:
- book_triage.bat - Easy launcher (double-click to start)
- start.bat - Alternative launcher with detailed output
- install.bat - Installs required dependencies
- examples/sample_books.csv - Sample book database

For more information, see README.md
